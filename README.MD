# Tic Tac Toe

This is a web-based Tic Tac Toe game with multiple game modes, including Player vs Player, AI opponents, and various board sizes.

## Features

- **Player vs Player**: Play against another human player.
- **AI Opponents**: Play against different levels of AI, including Smart AI, Easy AI, and Random AI.
- **Ultimate Mode**: A more complex version of Tic Tac Toe with a 3x3 grid of 3x3 grids.
- **Mega Modes**: Play on larger boards (4x4 and 5x5) with different win conditions.
- **Inverted Mode**: Try to lose the game instead of winning.
- **Invisible Mode**: The board becomes invisible after a few moves, testing your memory.

## Installation

1. Clone the repository or download the ZIP file.
2. Open the `index.html` file in your web browser.

## Usage

1. Open the game in your web browser.
2. Select a game mode from the dropdown menu.
3. Play the game by clicking on the cells in the grid.
4. The status message will display whose turn it is and the game result.

## Game Modes

- **Player vs Player**: Two players take turns to place their marks (X and O) on the board.
- **Smart AI**: Play against an AI that uses a minimax algorithm to make optimal moves.
- **Easy AI**: Play against an AI that makes random moves with a slight preference for winning moves.
- **Random AI**: Play against an AI that makes completely random moves.
- **Inverted Mode**: The goal is to lose the game instead of winning.
- **Invisible Mode**: The board becomes invisible after a few moves, testing your memory.
- **Mega 4x4 PvP**: Play on a 4x4 board with another player.
- **Mega 5x5 PvP**: Play on a 5x5 board with another player.
- **Ultimate Mode**: A 3x3 grid of 3x3 grids, where you need to win the smaller grids to win the game.

## Customization

You can customize the game by editing the `index.html` and `style.css` files. The game logic is implemented in the JavaScript section of the `index.html` file.
